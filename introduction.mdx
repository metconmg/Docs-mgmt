The MCMG Simulation Knowledge Base is the single source of truth for everyone working on or with the simulation platform. Whether you are designing a new scenario, authoring simulation entities, validating outputs, or reviewing artifacts produced by the project, this knowledge base gives you the concepts, standards, and step-by-step guidance you need.

## How to Use This Knowledge Base

<Steps>
  <Step title="Read the Project Scope">
    Start with [Project Scope](/project-scope) to understand what the simulation covers, who the stakeholders are, and what success looks like.
  </Step>
  <Step title="Learn the Core Concepts">
    Work through the [Core Concepts](/concepts/simulation-fundamentals) section to build a shared mental model before diving into specifics.
  </Step>
  <Step title="Follow the Design Standards">
    Before creating any artifact — scenario, entity, or configuration file — review the [Design Standards](/standards/naming-conventions) to ensure your work is consistent with the rest of the project.
  </Step>
  <Step title="Use the Guides">
    The [Development Guides](/guides/environment-setup) walk you through practical tasks end-to-end. Start with environment setup, then move to building and running simulations.
  </Step>
</Steps>

<Tip>
  If you encounter an unfamiliar term, check the [Glossary](/reference/glossary) first. The MCMG project uses precise terminology that may differ from general simulation industry usage.
</Tip>
